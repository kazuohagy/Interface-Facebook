## Comandos para usar no terminal :computer:

### `npm install @mui/stlyes`

Para baixar os estilos do mui

### `npm install @mui/icons-material`

Para baixar o icone do botão :suspect:

### `npm install `

Para pegar o react.\

### `npm start `

Para iniciar o react no local host.\
